安装SDAutoLayout的时候的错误

今天在安装SDAutoLayout的时候出现各种错误，所以在此记录一下:

  • 增加’pod ‘SDAutoLayout’, \’~> 2.1.2\’’出错

解决:

因为我用了：source ‘https://gitcafe.com/akuandev/Specs.git'，这个源，貌似这个源倒闭了。
我去掉这个源，就正常了。

  • 编译工程出错

解决:

  • 收先关闭xcode 然后pod install,如果不行就先删除xcworkspace,Podfile.lock,Pods文件夹,rm -rf ~/Library/Developer/Xcode/DerivedData/*,然后 pod install
  • pod deintegrate –verbose 　　
    pod install –verbose
