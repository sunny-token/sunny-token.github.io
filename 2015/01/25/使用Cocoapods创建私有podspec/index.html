使用Cocoapods创建私有podspec

发表于 2015-01-25   |   分类于 ios

整体先说明一下创建一个私有的podspec包括如下那么几个步骤：

  • 安装cocoapods（这个步骤省略，网上搜，一大堆）。cocoapod安装
  • 创建Pod所对应的podspec文件。
    运行: pod spec create TZImagePickController.podspec , 创建好spec文件
  • 打开TZImagePickController.podspec
    vim，sublime等等,删除注释，留下自己需要的。
  • 本地测试配置好的podspec文件是否可用。
    运行：pod lib lint ，检测spec文件是否符合规则。

成功则会出现以下数据，反之有warning或者error，就需要修改。

在个人项目中的Podfile中增加刚刚制作的好的Pod并使用。
在podfile里面添加：pod ‘TZImagePickController’, :path => ‘../ft_common/TZImagePickController.podspec’
然后cd到工程目录下，运行pod install ，到此就成功了！
